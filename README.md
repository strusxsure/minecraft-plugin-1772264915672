# LifeSteal Plugin for Spigot/Paper 1.21

A Minecraft plugin that implements lifesteal mechanics where players gain health from killing others.

## Features
- **Heart Theft**: Kill players to steal their max health
- **Health Limits**: Min 1 heart (2 HP), Max 20 hearts (40 HP)
- **Elimination**: Players reaching 1 heart are eliminated (spectator mode)
- **Heart Item**: Craftable heart item using Nether Star + Gold Blocks
- **Withdraw Command**: Convert current health to Heart items
- **Persistence**: Health data saved across restarts

## Commands
- `/lifesteal withdraw <amount>` - Convert health to Heart items
